> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans (inline). Steps use checkbox (`- [ ]`) tracking.

**Goal:** Build the full 8-section Alive Spectra homepage (Refraction hero + proof, services, plans, ecosystem, projects, founder, CTA) on typed data files with Organization/LocalBusiness JSON-LD.

**Architecture:** `components/home/` section components assembled in `app/page.tsx`; content in `data/*.ts`; client logos in `public/logos/`; JSON-LD in `lib/json-ld.ts`. Reuses tokens, primitives, `Reveal`.

- Client logos monochrome (grayscale → color on hover); meaningful `alt`; **ICRC held** (`holdForPermission`).
- Verify per task: `type-check` + `lint` + `build` clean; final confirmed on `/`. Commit per task (branch `feat/homepage`).

### Task 3: JSON-LD

**Produces:** `organizationLd()`, `localBusinessLd()` → JSON objects from `siteConfig`.

- [ ] **Step 1:** Create `lib/json-ld.ts`:

```ts
import { siteConfig } from "@/lib/site";

export function organizationLd() {
  const { name, url, founder, contact } = siteConfig;
  return {
    "@context": "https://schema.org",
    "@type": "Organization",
    name,
    url,
    foundingDate: String(siteConfig.founded),
    founder: { "@type": "Person", name: founder },
    email: contact.email,
    telephone: contact.phones,
    address: {
      "@type": "PostalAddress",
      streetAddress: contact.address.line,
      addressLocality: contact.address.city,
      postalCode: contact.address.postalCode,
      addressCountry: contact.address.country,
    },
  };
}

export function localBusinessLd() {
  return {
    ...organizationLd(),
    "@type": "ProfessionalService",
    areaServed: "Bangladesh",
    slogan: siteConfig.tagline,
  };
}
```

- [ ] **Step 2:** type-check clean. Commit.
